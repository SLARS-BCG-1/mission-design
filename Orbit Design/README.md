# SLARS-BCG-1 Orbit Design Files

This repository contains open orbit design files for **SLARS-BCG-1**, Sri Lanka’s first school CubeSat mission.  
The files are prepared for use with **GMAT (General Mission Analysis Tool)**, NASA’s open-source mission design and trajectory simulation software.

---

## 📂 Contents
- Orbit design parameter files (`.script`)
- Coverage and access analysis configurations
- Supporting documentation for mission design validation

---

## 🚀 Viewing Orbit Design Files in GMAT

1. **Download GMAT**  
   Official NASA distribution: [Download GMAT](https://software.nasa.gov/software/GSC-18094-1)

2. **Install GMAT**  
   Follow the installation instructions provided on the NASA software page.

3. **Open Orbit Design Files**  
   - Launch GMAT.  
   - Go to **File → Open Script…**.  
   - Select the `.script` file from this repository.  
   - Run the simulation to visualize orbit design and coverage.

4. **Validate Outputs**  
   Compare simulation results with mission requirements to ensure locked parameters match expected outcomes.

---

## 📖 Notes
- GMAT is cross-platform (Windows, macOS, Linux).  
- Files are configured for educational and competition use, highlighting the historic nature of SLARS-BCG-1.  
- Ensure you are using the latest GMAT release for compatibility.
